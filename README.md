# EasyLeaderstatsCreate
Eng   This repository is a library for the Lua programming language in Roblox Studio. This library will simplify the creation of leaderstats. You just need to write one line of code and to create currency in one line of code you can also interact with DataStore2.
Rus   Этот репозиторий предстовляет собой библеотеку для языка програмирования Lua в Roblox Studio. Эта библеотека упростит создание leaderstats`ов. Вам всего лишь нужно написать одну строчку кода и для создания валюты тоже одна строка кода также можно взаимодействовать с DataStore2.
